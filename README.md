
Procedural TypeScript command line text adventure.

## Prerequisites

- Node.js (ESM)
- npm
- (optional) `make` for the Makefile workflow

## Dev

```bash
npm install
npm run dev
```

Or, with `make`:

```bash
make setup
make dev
```

## Build + run

```bash
npm run build
npm start
```

Or, with `make`:

```bash
make build
make start
```
